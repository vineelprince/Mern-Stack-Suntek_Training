## Backend Development

1. Create git repo
    git init

2. Add .gitignore file

3. Create .env file to store environment variables and read data from it using "dotenv" file
    npm install dotenv

4. Generate package.json file
    npm init -y

5. Create Express server
    npm install express

6. Connect to database
    npm install mongoose

7. Add middlewares (like body-parser, error-handling)
    
8. Design Schemas and create models

9. Design REST API for all resources
